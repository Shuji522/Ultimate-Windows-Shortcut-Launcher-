# Ultimate Windows Shortcut Launcher
A fast, lightweight Chrome extension designed to help you master Windows keyboard shortcuts. Never forget a hotkey again with instant search, category filtering, and favorite-saving features!

# 🚀 Features
Quick Search: Find any shortcut in milliseconds.

Category Filtering: Easily browse shortcuts by System, Navigation, File Explorer, and more.

Favorites: Save your most-used hotkeys for quick access.

Dark Mode: Easy on the eyes for late-night coding or browsing.

One-Click Copy: Click any shortcut to instantly copy the keys to your clipboard.

# 🛠️ How to Install
Download this file as a ZIP file.

Unzip the folder to a location on your computer.

Open Chrome and go to chrome://extensions/.

Enable Developer mode (toggle in the top-right corner).

Click Load unpacked and select the folder you just unzipped.

Done! Click the puzzle piece in your browser toolbar to start using it.

# 📝 How to Use
Click to Copy: Click any shortcut in the list to copy it to your clipboard.

Shift + Click: Hold Shift while clicking a shortcut to add/remove it from your Favorites.

Filter: Use the dropdown menu to jump between categories.

# 💻 Tech Stack
HTML5 / CSS3: For a clean, responsive user interface.

JavaScript (Vanilla): For logic, filtering, and search functionality.

Chrome Storage API: To save your favorite shortcuts locally.
